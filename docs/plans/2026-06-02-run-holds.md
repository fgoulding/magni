# Program Run Holds Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Let a user pause one active program run for a dated window without advancing scheduled workouts or changing other active runs.

**Architecture:** Add a `program_run_holds` table keyed to `program_runs`. Service helpers create/delete active hold windows and expose hold state to Today, Programs, and Calendar. Schedule projection skips hold dates before consuming workout slots, so the program shifts forward instead of silently progressing.

**Tech Stack:** Next.js App Router route handlers, React server pages, client form components, better-sqlite3, Vitest, Playwright.
